# open-data-api
공공데이터 포털 API 모음집

이 저장소는 정부에서 공개하는 공공데이터 포털을 조금더 쉽게 사용하기 위해여 요약한 곳입니다.
부동산관련 API가 그 대상이며 중구난방으로 작성되어 있는 문서가 한번에 정리되었으면 합니다.

자료의 크기상 모든 API를 정리 할수 없고 제가 몸담고 있는 부동산 위주로 정리를 시작하려 합니다.

첫번째로 **"국토부 실거래가"** 를 정히다록 하겠습니다.
작성한 자료는 open-data-api에서 작업하고 완료되면 다른 저장소로 이동하여 pip를 통해 install 까지 할수 있는 것을 목표로 만들겠습니다.
